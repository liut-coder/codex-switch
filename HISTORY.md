开发历史

版本 3.0.0 - 2026-04-29

重大重构
- 重写 `codex-switch.sh`，修复原脚本残缺无法执行的问题
- 将配置管理状态迁移到 `~/.config/codex-switch/`
- 切换配置时自动同步 shell 环境和官方 `codex` CLI
- 新增独立 `install.sh`，支持仓库安装

新增功能
- `--activate` 仅切换不启动
- `--show` 查看当前或指定配置详情
- `--add` / `--delete` 命令行管理配置
- 自动写入 `~/.bashrc` / `~/.zshrc` 的受管区块

兼容性调整
- 保留 `codex-switch` 和 `sw` 两个命令入口
- 当未安装官方 `codex` CLI 时，回退写入 `~/.config/codex/config.json`

版本 2.0 - 2026-04-30

重大更新
- 集成所有功能到单一脚本：不再需要手动调用多个脚本
- 自动联动机制：切换配置自动同步到官方 Codex CLI
- 一键启动：配置完成后自动启动 Codex
- 快捷键 `sw`：最快的配置切换方式

版本 1.0 - 2026-04-29

初始功能
- 基础配置管理（增删改查）
- 多配置切换
- 连接测试
- 快速配置模板
